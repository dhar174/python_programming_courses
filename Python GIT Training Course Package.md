**Course Description:**

This comprehensive 96-hour Python Programming course equips students with the skills to design, develop, and deploy Python applications across multiple domains. Starting from the basics of Python syntax, variables, loops, and functions, students gradually advance to mastering object-oriented programming, GUI design, web services, databases, data science, and deployment techniques.

Through hands-on labs, real-world scenarios, and capstone projects, learners will build scalable and secure applications while preparing for globally recognized certification **Python Institute’s PCEP/PCAP**. acks.

**Course Objectives:**

By the end of this course, learners will be able to:

**Module 1 – Python Programming Fundamentals**

* Set up and configure a Python development environment.
* Write Python scripts using variables, operators, data types, and control flow.
* Apply data structures (lists, dictionaries, tuples, sets).
* Use functions, modules, and basic OOP.
* Perform file I/O and error handling.
* Build command-line programs and prep for PCEP certification.

**Module 2 – Advanced Python Development**

* Apply advanced object-oriented design patterns and classes.
* Build GUI applications using frameworks like Tkinter or PyQt.
* Connect and manage SQL databases in Python.
* Develop and secure RESTful web APIs using Flask/Django.
* Analyze and visualize data; implement ML regression.
* Use version control, unit testing, and package deployment tools.
* Prepare for Python Institute PCAP exam.

**Target Audience:**

* Beginners looking to enter software or application development.
* Technical professionals transitioning into Python-based roles.
* Students preparing for Python certification exams (PCEP/PCAP).
* Developers interested in full-stack, automation, API, or data science roles.

**Prerequisites:**

* Basic computer skills (familiarity with Windows/macOS or Linux).
* For Advanced Module: Completion of Python Fundamentals module or equivalent experience.
* Optional: Prior exposure to programming logic or scripting is helpful.
