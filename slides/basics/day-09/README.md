## Learning Outcomes

By the end of Session 9, learners can:

### Hour 33 — Functions: def, parameters, return
- Write simple functions with parameters and return values
- Distinguish when to use `return` vs `print`
- Refactor repeated code into named functions

### Hour 34 — Scope + common function mistakes
- Explain local vs global scope
- Avoid relying on global variables inside functions
- Use default parameters for optional behavior

### Hour 35 — Functions with collections
- Pass lists and dicts into functions
- Decide when to mutate in-place vs return a new collection
- Document function expectations with docstrings

### Hour 36 — Modules: imports and creating utils.py
- Import from the Python standard library
- Create a personal utility module (`utils.py`)
- Avoid common import pitfalls (name conflicts, wrong working directory)
