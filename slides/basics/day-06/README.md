## Learning Outcomes

By the end of Session 6, learners can:

### Hour 21 — Choosing the Right Structure
- Select list vs set vs dict based on task requirements
- Explain tradeoffs in terms of clarity, duplicates, ordering, and lookup
- Refactor a solution to use a more appropriate structure

### Hour 22 — Drill Circuit
- Build short solutions with lists, sets, and dictionaries
- Practice reading task requirements before choosing a structure
- Debug small data-structure exercises under time pressure

### Hour 23 — Mini-Project
- Create an in-memory tracker using a dictionary of records
- Add, update, search, and summarize entries
- Format reports clearly with loops and string formatting

### Hour 24 — Checkpoint 3
- Build a simple contacts program with safe key access
- Explain the difference between `dict.get()` and `dict[key]`
- Add a basic search feature using loops and conditionals
