## Learning Outcomes

By the end of Session 8, learners can:

### Hour 29 - Menu Loops
- Build a menu-driven `while` loop that repeats until quit
- Route user choices with `if/elif/else`
- Keep shared program state in memory across actions

### Hour 30 - Input Validation
- Use `.isdigit()` and simple guards before conversion
- Re-prompt when input is invalid
- Explain the limits of beginner-friendly validation patterns

### Hour 31 - CLI Contact Manager
- Combine loops and data structures into a working CLI program
- Implement add, list, search, and delete features
- Handle missing-name cases without crashing

### Hour 32 - Checkpoint 4
- Build a coherent CLI program using loops, conditionals, and data structures
- Apply basic validation for empty names and invalid task numbers
- Test normal cases and boundary cases intentionally
