## Learning Outcomes

By the end of Session 7, learners can:

### Hour 25 — Conditionals: if/elif/else and Boundaries
- Write clear conditional logic with properly ordered branches
- Handle boundary values correctly in multi-tier decisions
- Choose flat if/elif/else over nested conditionals for readability

### Hour 26 — while Loops + Sentinel Patterns
- Use while loops for repeated prompts and user-driven repetition
- Implement sentinel value patterns (e.g., 'q' to quit)
- Use break and continue appropriately
- Avoid common infinite loop causes

### Hour 27 — for Loops + range()
- Use for loops with range() in all three forms
- Explain inclusive start / exclusive stop in range
- Choose between for and while loops based on the situation

### Hour 28 — Loop Patterns: Counters, Accumulators, Min/Max
- Apply counter, accumulator, and min/max patterns reliably
- Initialize min/max safely (not to 0)
- Track running totals and extremes through a loop
