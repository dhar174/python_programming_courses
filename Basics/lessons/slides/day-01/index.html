Day breakdown

Use this page as a quick orientation before opening the full slide deck. The four hour blocks mirror the instructor runbook sequence.

Hour 1

Environment setup

Install or open the classroom Python environment, confirm the interpreter works, and practice the edit-run-debug loop.

Hour 2

Printing and literals

Use print(), string literals, numbers, comments, and predictable output formatting.

Hour 3

Variables and basic types

Store values with clear names and distinguish strings, integers, floats, and booleans.

Hour 4

Operators and expressions

Combine values with arithmetic, assignment, and comparison-friendly expression patterns.

By the end of this day

  • Run Python code in the classroom environment.
  • Use print() to produce deterministic output.
  • Create and update variables with clear names.
  • Evaluate simple expressions with common operators.
