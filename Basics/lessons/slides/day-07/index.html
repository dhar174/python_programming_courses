Day breakdown

Use this page as a quick orientation before opening the full slide deck. The four hour blocks mirror the instructor runbook sequence.

Hour 25

if/elif/else

Branch program behavior based on conditions.

Hour 26

while loops + sentinel patterns

Repeat work until a condition changes using controlled sentinel values.

Hour 27

for loops + range()

Iterate over sequences and use range() to drive counting loops.

Hour 28

Loop patterns

Apply counters, accumulators, and min/max patterns to real problems.

By the end of this day

  • Write clear conditional branches.
  • Build terminating while loops with sentinel values.
  • Use for loops and range() to iterate sequences.
  • Apply counter and accumulator loop patterns.
