### For Instructors
1. Open `slides/day-01-session-1.html` in any modern browser
2. Press F11 for full-screen presentation mode
3. Navigate with arrow keys or Space
4. Use progress bar to track pacing
5. Code examples can be copied directly from slides
6. Print to PDF for offline reference if needed

### For Students
1. Open `slides/index.html` to see all available decks
2. Click on "Basics Day 1 — Session 1" to access slides
3. Navigate at your own pace
4. Bookmark specific slides via URL (e.g., `#15`)

  - Colors (background, text, accent)
  - Fonts (body, heading, code)
  - Spacing and padding
  - Animation speed
- No build process required - just edit and reload
